# B站房管工具

一个用于B站直播间的房管辅助工具，支持实时弹幕监控、敏感词过滤、用户禁言等功能。当房管太费眼睛了，网上的项目根本不好使，自己手搓一个简单版，自用。

## 功能特性

- 🔴 **实时弹幕监控** - WebSocket连接，多服务器冗余，低延迟接收弹幕
- 💎 **醒目留言** - Super Chat 实时显示，带颜色标识
- 🛡️ **敏感词过滤** - 支持自定义敏感词库（.md文件），自动禁言
- 🔨 **用户管理** - 禁言/解禁用户，查看禁言列表
- 👥 **舰队标识** - 总督/提督/舰长特殊背景标识
- 📊 **数据统计** - 在线用户、弹幕数量实时统计
- 🎨 **界面定制** - 三档字体大小调整
- 🔄 **智能重连** - 指数退避重连，连接更稳定

## 快速开始

### 方式一：使用预编译 EXE（推荐）

适用于 Windows 用户，无需安装 Python。

1. **下载** `B站房管工具-v1.1.0.zip` 并解压
2. **配置 Cookie**
   - 复制 `.env.example` 为 `.env`
   - 填写 `SESSDATA` 和 `BILI_JCT`（获取方式见下方）
3. **（可选）配置敏感词**
   - 编辑 `sensitive_words/` 目录下的 `.md` 文件
4. **启动**
   - 双击 `B站房管工具.exe`
   - 浏览器自动打开 `http://127.0.0.1:8000`

### 方式二：源码运行

适用于开发者或需要自定义的用户。

#### 环境要求

- Python 3.10+
- Windows/Linux/macOS

#### 安装依赖

```bash
pip install -r requirements.txt
```

#### 配置Cookie

1. 登录B站网页版
2. 按F12打开开发者工具 → Application/应用 → Cookies
3. 复制 `SESSDATA` 和 `bili_jct` 的值
4. 编辑 `.env` 文件：

```env
SESSDATA=你的SESSDATA值
BILI_JCT=你的bili_jct值
```

### 准备敏感词文件（可选）

在 `sensitive_words/` 目录创建 `.md` 文件：

```markdown
# 敏感词列表
广告
推广
加群
QQ群
```

### 准备舰队背景图（可选）

舰队背景图已内置，如需自定义：

**源码版本**: 替换 `app/static/guard_bg/` 目录下的图片
**EXE 版本**: 需重新打包

### 启动服务

**EXE 版本**: 双击 `B站房管工具.exe`

**源码版本**:

```bash
python run.py
```

浏览器会自动打开 `http://127.0.0.1:8000`

## 使用说明

### 连接直播间

1. 在顶部输入框输入直播间房间号
2. 点击"开始监听"按钮
3. 连接成功后即可看到实时弹幕

### 禁言用户

**方式一**：鼠标悬停在弹幕上，点击"禁言"按钮

**方式二**：点击用户名打开菜单，选择禁言

**方式三**：在"最近发言"列表中点击禁言

**禁言时长选项**：

- 当前直播
- 1小时
- 24小时
- 7天
- 30天
- 永久

### 敏感词过滤

1. 在右侧"敏感词过滤"面板选择敏感词库文件
2. 勾选"自动禁言含敏感词的用户"开启自动禁言
3. 系统会自动检测弹幕并禁言发送敏感词的用户

### 解禁用户

1. 在"禁言列表"中查看被禁言用户
2. 点击"解禁"按钮（有二次确认）

### 调整字体大小

点击弹幕区域右上角的"字体: 小/中/大"按钮切换

## 注意事项

1. **Cookie安全**：`SESSDATA` 相当于登录凭证，请勿泄露
2. **权限要求**：必须是直播间的房管才能禁言用户
3. **频率限制**：频繁操作可能触发B站风控
4. **弹幕遗漏**：高并发时可能有少量弹幕遗漏（B站服务端限制）

## 技术栈

- **后端**：FastAPI + WebSocket + httpx
- **前端**：Vue 3 (CDN) + 原生CSS
- **协议**：B站直播弹幕WebSocket协议
